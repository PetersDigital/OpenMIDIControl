## 2025-03-26 - [Tooltip Context]
**Learning:** In Flutter, icon-only interactive elements like `IconButton` or `GestureDetector` containing an `Icon` should always be wrapped in a `Tooltip` to provide accessible labels for screen readers and helpful context on desktop hovers.
**Action:** Always verify if an `IconButton` or standalone icon interaction component has a descriptive text equivalent or tooltip attached before proceeding with an accessibility enhancement.
